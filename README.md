# Fiona Lim Portfolio

A personal portfolio website for Fiona Lim, showcasing digital operations, system design, GenAI workflow automation, and selected project work.

The site is built as a static website using HTML, CSS, JavaScript, and Bootstrap 5. It is designed for recruiters, hiring managers, and collaborators who want to understand Fiona's experience, working style, technical projects, and design thinking process.

## Pages

- `Home`: introduction, core positioning, featured work, and quick portfolio highlights.
- `Design Thinking`: breakdown of the website planning process, UX decisions, heuristic analysis, and responsive design considerations.
- `About`: background, work experience, education, skills, and interests.
- `Portfolio`: project showcase, demo videos, contact details, resume download, and contact form.

## Features

- Responsive layout across desktop, tablet, and mobile.
- Bootstrap 5 navigation, grid system, modals, and components.
- Custom CSS visual system with dark theme styling.
- JavaScript interactions including typewriter text, scroll reveal, active navigation, counters, video modal handling, and contact form feedback.
- Formspree-powered contact form.

## Local Preview

Open `docs/index.html` directly in a browser, or serve the `docs/` folder with any static file server.

## Configuration

Local-only environment values should be stored in `.env`. Do not commit `.env` to the repository.

Use `.env.example` as the reference for required variable names.
